# WebWidgets - Component Library

[WebWidgets](https://web-widgets-v1.netlify.app/) - A ReactJS Component Library focused on developing resuable components.

## Preview

## Components

WebWidgets contains the following components:

- Avatar
- Alert
- Badge
  - Badge on Icons
  - Badge on Avatars
- Button
  - Primary Buttons
  - Secondary Buttons
  - Link Button
  - Icon Button
  - Floating Action Button
- Card
  - Vertical Card
  - Card with Badge
  - Card with Dismiss
  - Card with Text Overlay
  - Text only Card
  - Horizontal Card
  - Card with Shadow
- Headings
- Text
- Image
  - Responsive Image
  - Round Image

## Built with

- ReactJS
- Vanilla CSS
- React-Syntax-Highlighter
